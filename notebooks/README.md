# Jupyter Notebooks (`notebooks/`)

This directory contains Jupyter notebooks used for testing, debugging, analysis, and demonstrating the functionality of the IRIS system.

## Purpose

Notebooks provide an interactive environment for running parts of the IRIS pipeline, experimenting with queries, inspecting intermediate results, and performing specific development or analysis tasks.

## Notebooks

*   **`db_export_standalone.ipynb` / `db_export_tool.ipynb`**: These notebooks likely contain code for exporting data from the database, perhaps for backup, analysis, or sharing purposes. One might be a standalone script, while the other could be designed as a reusable tool.
*   **`debug_notebook.ipynb`**: A general-purpose notebook for debugging specific issues or testing components of the IRIS system in isolation.
*   **`process_monitor_analysis.ipynb`**: Used for analyzing data generated by the `process_monitor.py` module (located in `iris/src/initial_setup/`), potentially visualizing process performance or status logs.
*   **`test_notebook.ipynb`**: This appears to be the primary notebook for end-to-end testing of the IRIS query pipeline. As mentioned in the main project README, users can modify the `conversation` variable here to input queries and run the cells to see the system's response.

## Usage

To use these notebooks:

1.  Ensure you have Jupyter Notebook or JupyterLab installed in your project's virtual environment (`pip install notebook` or `pip install jupyterlab`).
2.  Activate your virtual environment.
3.  Navigate to the `iris-project` root directory in your terminal.
4.  Run `jupyter notebook` or `jupyter lab`.
5.  Open the desired notebook file from the Jupyter interface in your browser.
