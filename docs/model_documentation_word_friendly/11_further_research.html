Section 11: Further Research

Based on our development and testing of the IRIS system, we have identified several key areas for further research:

  1. On-Premises Model Integration - Explore the use of on-premises LLM alternatives to replace the current dependency on GPT models, particularly focusing on models that can be deployed within RBC's security perimeter.
  2. Advanced Vision Model Integration - Test alternative vision models to improve the preprocessing of infographics in financial documents, enhancing the system's ability to extract and represent visual information.
  3. Complex Table Preservation - Investigate methods for maintaining the structure and relationships in complex tables when converting from source documents to usable formats within the IRIS system.
  4. Markdown Conversion Alternatives - Evaluate alternatives to Azure Document Intelligence for converting documents to markdown, looking at both accuracy improvements and cost efficiency.
  5. Alternative Retrieval Methods - Test new retrieval approaches beyond the current catalog and semantic search methods, particularly exploring hybrid retrievers and re-ranking approaches.
  6. Multilingual Capabilities - Assess and implement support for multilingual queries and document ingestion, expanding IRIS's accessibility to non-English financial documentation and user queries.

These research directions will be prioritized based on business requirements and technical feasibility, with implementation planned over the next 12-18 months.
