IRIS Project - Data Dictionary

1. Data Terminology, Acronyms, and Abbreviations

Term/Acronym Definition
Agent Architecture A modular system design pattern where specialized components (agents) perform specific tasks in the processing pipeline
Agent Clarifier Component that assesses query context sufficiency and creates research statements
Agent Direct Response Component that generates responses using only conversation history
Agent Planner Component that selects optimal databases for query execution
Agent Router Component that determines processing path based on query analysis
Agent Summarizer Component that integrates findings from multiple sources into coherent, properly cited responses
Catalog Search - Excel Specialized pipeline for processing Excel spreadsheets containing tabular data
Catalog Search - Large Retrieval method for large documents with distinct sections, implementing section-level selection
Catalog Search - Small Standard retrieval method for documents with simple structures, processing entire documents as single units
Catalog Search - Vision Specialized pipeline for processing infographics and visual content
Conversation History Complete record of user-system interactions in a session
Database Subagents Components that retrieve and pre-process information from various knowledge sources
Direct Response Path Processing pathway for queries answerable using only conversation history
Document Refresh Pipeline Process that runs on a scheduled basis to process finance inputs and transform them into database content
GPT-4o Large language model used for complex reasoning tasks and final response generation
GPT-4o-mini Large language model used for efficient processing of routine decisions
Knowledge Boundaries Limitations defining what information the model can access and provide
Maven User interface platform that hosts the IRIS system
Metadata Search Query mode returning only document names and descriptions
Process Monitoring System tracking execution stages, token usage, and performance metrics
Qwen2 7B VL Vision language model used for extracting text and visual elements from image content
RAG (Retrieval-Augmented Generation) Framework combining retrieval of information with language model generation to produce grounded responses
Research Path Processing pathway for complex queries requiring database research
Semantic Search - Large Vector-based similarity search methodology for reference documents
text-embedding-3-large Model used for generating vector representations (embeddings) in semantic search
Token-Aware Chunking Algorithm that segments documents by optimizing information density while preserving semantic structure
